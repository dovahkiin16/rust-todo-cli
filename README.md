# RUST TO-DO CLI
A CLI based to-do app made using rust.

## Usage

See all To-dos
```
todo-cli get-all
```

Create new To-do
```
todo-cli add "this is my new to-do"
```

Mark task as done
```
todo-cli done 1
```

Delete todo
```
todo-cli remove 1
```
